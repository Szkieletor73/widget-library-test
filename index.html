In order to initiate the widget tree, click the "Init" button on the element you wish to be the root initiator.

Similarly, use the "Destroy" button on the root destroyer element to destroy widgets.

Base widget classes have been applied - green are initialized widgets, yellow are loading widgets, and red are destroyed widgets.

'widget-destroyed' class lingers after widgets have been destroyed - this allows checking for previously active widgets that were destroyed.
